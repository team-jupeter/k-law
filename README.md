# K-Law — AI 기반 법률 중재 보조 시스템

> **"모든 시민이 평등하게 법의 보호를 받을 수 있는 사회"**  
> K-Law는 AI를 활용하여 대법원 판결 수준의 법리 분석을 누구나 빠르고 저렴하게 이용할 수 있도록 설계된 오픈소스 중재 보조 방법론입니다.

## K-Law란?

K-Law는 대한민국 헌법·법률·대법원 판례를 기반으로 분쟁 사건을 분석하고, 중재인의 독립적 판단을 보조하는 **AI 법률 중재 방법론**입니다.

- **입력**: 분쟁 당사자 간 합의된 사실관계 (A4 0.5~1장)
- **출력**: 법리 분석 보고서 + 예상 판결 방향 + 초등학생도 이해하는 해설
- **검증**: 80건의 실제 대법원 판결과 비교 검증 완료

### 작동 원리

```
사실관계 입력 (A4 0.5~1장)
        ↓
K-Law 원리 체계 적용 (v6.3, 원리 0~20)
        ↓
헌법·법률·판례 기반 법리 분석 (1~5분)
        ↓
중재인 독립적 검토 + 당사자 의견 수렴
        ↓
중재 판정 (법적 구속력)
```

## 법적 근거

K-Law는 대한민국 **중재법 제8조**에 따른 서면 중재합의를 기반으로 운영됩니다.

- 양 당사자의 서면 동의를 전제로 하며, AI는 중재인 보조 도구로만 기능
- 최종 판정은 **자연인 변호사 중재인**이 독립적으로 내림
- 헌법 제27조(재판청구권)와 충돌하지 않는 임의적 분쟁 해결 절차
